This menu item allows you to see any open actions and create associated csv extract files.

For all options, the data selected for the extract is shown in preview with buttons to create the extract file and to close the action.

New member packs Shows new members since last batch. Create a mail merge file for generating new members certificates etc.
Renewal cards Shows members who have renewed since last batch. Create a mail merge file for covering letter when sending cards
Other Shows members with any other open actions
Selecting one of the options will show a list of all members with that action open.

 

When the list of members for that action is shown, there are two options:

Extract Create the mail merge file
Close Set the relevant action for these members to Closed
