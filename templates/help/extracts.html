Extracts | Custom

This menu option allows you to create an extract of the database

  1. Go to membership.thedonstrust.org
  2. Click on "Extracts"
  3. Click on "Custom" from the drop-down list

You will see the following screen

The first section allows you to search for members by giving values in any of the fields. The values shown in this example will select all active Juniors aged 12-16 who accept email communications.
The second section allows you to specify the fields to extract.

When finished, click on Submit. The data will be retrieved and shown as follows:

Click on Extract to create a csv file.
